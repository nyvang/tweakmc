Shortcuts in TweakMC

To lighten the daily workload a little, the system is designed to use some keyboard shortcuts, for those who do not wish to use the mouse all the time. The keyboard shortcuts are as follows:


Key Value
F1 This help menu
F2 Calculator
F3 E-mail helpdesk
CTRL+L Logout
CTRL+O Order tab
CTRL+V Vehicle tab
CTRL+N Owner tab
CTRL+W Workstations tab


In future releases, we plan to implement more shortcuts as they are proven to be more effecient and a much better ergonomic solution than the mouse.

If You have any suggestions regarding new shortcuts that You find useful, please dont hesitate to contact us.
