TweakMC

What is this?

The TeaN TALC "TweakMC" (henceforth refered to as "the system") is designed for KJ Motorcycles A/S (KJMC) as a schoolproject during 2nd semester of Advanced Computer Studies, Computer Science.

The system is designed to help gathering information about the vehicles that are beeing handled at their workshop. Some of the key features are:

  • Storing data in a database

  • Presenting the data for customers

  • Error reporting via e-mail

  • Autocomplete

This was a very brief introduction to the system.
